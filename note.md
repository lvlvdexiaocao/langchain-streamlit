首次创建项目：

1、GitHub创建新仓库后，本地文件中

```
1.初始化文件夹
git init 

2.给远程仓库取别名
git remote add origin(别名，自定义) 远程仓库地址

3.拉取仓库文件至本地
git clone 远程仓库地址

4.添加本md文档测试（可选）
git add.
git commit -m "添加笔记，测试连接"
git push
```



2、使用pycharm打开，创建项目

```
1、使用conda创建虚拟环境
conda create -n langchain-streamlit python==3.11
2、激活虚拟环境
conda activate langchain-streamlit
3、pycharm中激活conda环境
```



3、项目初始配置

```
1、添加.gitignore文件，加入环境配置等
```
